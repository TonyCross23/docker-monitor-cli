# Docker Resource Monitor (Go)

A simple CLI tool to monitor Docker container CPU and RAM usage in real-time.

## Features
- Real-time CPU % and RAM % tracking.
- Clean table output for multiple containers.
- Cross-platform support (Windows, Mac, Linux).

## Installation
1. Go to the **Releases** page.
2. Download the binary for your OS.
3. **Linux/Mac**: Run `chmod +x <filename>` and move it to `/usr/local/bin`.
4. **Windows**: Run the `.exe` file in PowerShell or CMD.

## Pre-requisites
- Docker must be installed and running on your system.
